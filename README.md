# My Project

This is a Flutter frontend project with a Node.js backend.

## Setup Backend

1. Navigate to the backend folder and install dependencies:

```bash
cd backend
npm install
```

Create a .env file in the backend folder with the required environment variables.
Example:

# .env
```
PORT=urport
SUPABASE_URL=yoururl
SUPABASE_SERVICE_ROLE_KEY=yourkey
OPENAI_API_KEY=uropenaikey
```

Start the backend server:
```bash
npm start
```
